Documentation

This documentation provides an overview of the API endpoints and their usage.

  • GET /:shortCode: Redirects to the URL associated with the provided shortcode.
  • GET /api/preview/:shortCode: Returns a JSON object with details about the link associated with the provided shortcode.
  • GET /api/statistics/links: Fetches the total number of links.
  • GET /api/statistics/links/:shortcode: Fetches the number of clicks for a specific link.
  • POST /api/link: Creates a new link. Requires a JSON body with the "URL" field.
  • DELETE /api/link/:shortCode: Deletes a link associated with the provided shortcode.

CURL Requests

Create a new link:

curl -X POST -H "Content-Type: application/json" -d '{"URL":"http://example.com"}' https://nxt.mom/api/link
